# V39 Mechanism To Falsifiable Prediction Validation Protocol

V39 turns mechanism grammar into falsifiable operator-level predictions and
validates them against independent non-coordinate holdout evidence.

This is not atomistic structure prediction, molecular dynamics, native-metric
selection, or a folding-solved claim. Prediction sources and holdout validation
sources are kept separate. Holdout rows must not be coordinate-derived, runtime
reports, V38 answer-key artifacts, or placeholder citations.

Validation levels:

- `supported_by_independent_holdout`
- `partially_supported_clean_abstain`
- `contradicted_or_failed`
- `blocked_for_leakage`

V39 passes when all targets generate falsifiable predictions, at least two of
three targets are supported by independent holdout evidence, no target is
contradicted, all controls pass, and claim/MD/folding-solved flags remain
closed.
