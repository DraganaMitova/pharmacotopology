# V22 External Evidence and Annotation Acquisition Panel

V22 is a practical acquisition/readiness sprint. It consumes the locked V21 pressure-evidence panel and writes four core artifacts:

- `v22_external_evidence_manifest.json`
- `v22_annotation_preflight.json`
- `v22_coupling_availability_scan.json`
- `v22_next_target_decision.json`

It does not run MD, does not tune thresholds, does not select on native metrics, and does not upgrade any folding claim.

The selected V23 target is expected to be `p53_TAD_MDM2`, because the p53/MDM2 contrast has the strongest ready external-evidence path: isolated disorder reference context plus partner-bound interface evidence. KcsA and XCL1 remain deferred until local annotation/coupling gaps are closed.
